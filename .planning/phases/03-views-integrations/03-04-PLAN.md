**SMS24h API Client (ETP2-01)** — RESEARCH.md lines 292-303:
```javascript
async function smsAPI(action, extra = '') {
  const k = getSettings().sms_key;
  if (!k) throw new Error('Sem API key SMS24h. Configure primeiro.');
  const url = `https://api.sms24h.org/stubs/handler_api?api_key=${encodeURIComponent(k)}&action=${encodeURIComponent(action)}${extra}`;
  const r = await fetch(url, { method: 'GET' });
  if (!r.ok) throw new Error('HTTP ' + r.status);
  return await r.text(); // NOTE: Returns text, NOT JSON
}
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-17 | Tampering | etapa2.js — SMS24h API key in query string | accept | Key is in HTTPS request URL query string — encrypted in transit. Visible in browser DevTools network tab (single-user app, acceptable per ASVS L1). Proxy rewrites URL path but preserves query string. |
| T-03-18 | Tampering | etapa2.js — SMS response parsing | mitigate | Colon-delimited responses split with bounds check (parts.length >= 3 for ACCESS_NUMBER). Malformed responses caught by string checks (startsWith). No eval() on response text. |
| T-03-19 | Repudiation | etapa2.js — polling timer cleanup | mitigate | clearInterval called on ALL exit paths (STATUS_OK, timeout, cancel, confirm). Timer ID stored in etapa2State.timer for external cleanup. Previous interval always cleared before new. |
| T-03-20 | Information Disclosure | etapa2.js — phone number in DOM | accept | Phone number is a disposable virtual number purchased for verification. No PII. Displayed with copy button for user convenience. Single-user app. |
| T-03-21 | Denial of Service | etapa2.js — polling frequency | mitigate | Fixed 5-second interval. Server-side rate limiting (SMS24h) prevents abuse. 20-minute hard timeout prevents indefinite polling. Cancel button available. |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No packages installed. @noble/hashes loaded via dynamic import (same as ETP1-05). |
