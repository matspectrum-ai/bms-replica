    FILE 1: src/utils/format.js — Brazilian formatting utilities

    All functions are PURE (no side effects, no DOM access, no localStorage):

    1. fmtCNPJ(c): Formats CNPJ digits to ##.###.###/####-##.
       - Strip non-digits → .padStart(14,'0') → .slice(0,14)
       - Regex replace: /^(\d{2})(\d{3})(\d{3})(\d{4})(\d{2})$/ → '$1.$2.$3/$4-$5'
       - Edge case: null/empty → '00.000.000/0000-00' (padStart produces all-zeros)
       - Edge case: >14 digits → truncated to first 14

    2. fmtMoney(v): Formats number as Brazilian Real currency.
       - null or '' → return '—' (em dash)
       - Number(v) — if NaN, return original value as string
       - toLocaleString('pt-BR', {style:'currency', currency:'BRL', minimumFractionDigits:2})

    3. fmtDate(d): Formats date as DD/MM/AAAA.
       - Falsy (null, undefined, 0, '') → return '—'
       - new Date(d) — if invalid (isNaN), return original value as string
       - toLocaleDateString('pt-BR')

    4. formatBRPhone(num): Formats Brazilian phone number.
       - Convert to string, strip non-digits
       - If 13 digits starting with '55': strip prefix, recurse
       - If 11 digits: (XX) XXXXX-XXXX (mobile)
       - If 10 digits: (XX) XXXX-XXXX (landline)
       - Neither: return raw digits string

    5. calcAnos(inicio): Calculates years since inception date.
       - Falsy → return 1
       - Parse DD/MM/AAAA (regex match + Date constructor) or ISO format
       - Invalid date → return 1
       - Math.max(1, floor((today - date) / yearMs))

    EXPORT: export { fmtCNPJ, fmtMoney, fmtDate, formatBRPhone, calcAnos }

    FILE 2: src/utils/string.js — String manipulation utilities

    All functions are PURE (no side effects):

    1. onlyDigits(s): Removes all non-digit characters.
       - (s || '').replace(/\D/g, '')
       - Null → empty string

    2. slugify(s): Creates URL-friendly slug.
       - Null/undefined → 'empresa'
       - toLowerCase → normalize('NFD') → remove combining marks → replace non-alphanumeric → slice(0,28)
       - Empty result after cleanup → 'empresa' fallback

    3. escapeHTML(s): Escapes HTML special characters.
       - (s || '').toString()
       - Replace: & → &amp;, < → &lt;, > → &gt;, " → &quot;, ' → &#039;
       - Only these 5 entities (matches original — not full OWASP set)

    EXPORT: export { onlyDigits, slugify, escapeHTML }

    - Add extra validation beyond the original
